About

This is Workshops (still in a beta phase), is a project created at first to manage educational content in an easy and adaptable way from low income public schools (with low resource computers) to large courses with many students in an organization, company or a college compatible with each other, built under a free software license (this version is implemented with Django's technology).

Features
  • Free and Open Source
  • Easy to use
  • Easy instalation
  • Easy user registration
  • Allows upload your own courses or take courses made by the users
  • Allows import/export your own courses in a single file in a standard markdown/html5 format using the remark technology for course's presentation in a single text file.
